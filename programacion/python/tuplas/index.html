Definición

Una tupla es una coleccion de objetos que pueden tener diferentes tipos. Los datos se escriben entre parentisis, separados por comas como la siguinte sintaxis.

(dato1, dato2, dato3, ...... )

Los componentes de una tupla no se pueden modificar despues de haber sido creados.

Ejemplo:

x = ('Jaider', 43, 'Quintero', 09)

Se identifican desde la posicion 0, es decir

x[0]
x[1]
x[2:]

El mismo tratamiento de las listas.

x[0] = 'Edil' => esta asinacion no se puede, puesto que las tiplas no pueden cambiar su valor.
