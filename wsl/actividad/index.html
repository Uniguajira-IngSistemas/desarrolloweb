Actividad linux-Ubuntu

Creacion carpeta ambiente de trabajo

1). Entrar a Linux-Ubuntu Puede realizar este paso de dos maneras:
Opcion 1: Desde la busqueda de Windows, escribir para buscar la aplicacion Ubuntu y luego presione click sobre el resultado. escriba en la consola de linux-ubuntu:

pwd

y luego presione Enter. aparecera el siguiente resultado:

Opcion 2: Escriba en la busqueda de Windows, CMD y una vez que aparezca la consola de windows escriba el comando wsl y Enter. luego, una vez aparezca la consola de linux-ubuntu escriba: 

cd /home/ubuntu
pwd

Obteniendo el mismo resultado de la opción anterior:

2). Crear la estructuras de carpetas, de acuerdo a la siguiente imagen:

Si todo ha sido exitoso al momento de escribir el comando

pwd

debe estar en la ruta: /home/ubuntu


Para ver mejor esta estructura de arbor instalar la herramienta tree de la siguiente manera:


sudo apt-get update
sudo apt-get install tree

Puede ver la estructura de las carpetas y archivos creados, mediante

tree appsweb

Muestra el conttenido de la carpeta appsweb, incluyendo los archivos

o tambien 


tree /

Muestra la estructura de la raiz del sistema operativo, incluyndo carpetas y archivos

Tambien puede utilizar modalidades del comando tree, por ejemplo:


tree appsweb -d -L 3

Indicando que solo muestre las sub carpetas de appsweb hasta el nivel 3
